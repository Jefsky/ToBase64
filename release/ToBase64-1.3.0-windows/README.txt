ToBase64 v1.3.0
==============

一个简单的工具，用于将文件转换为Base64编码。

新功能:
1. 批量下载功能 - 可以选择特定文件进行批量下载
2. 默认保存路径设置 - 可以设置默认的保存路径，避免每次都需要选择
3. 文件勾选功能 - 可以通过复选框选择特定文件，支持全选/取消全选
4. 打开保存路径文件夹 - 可以直接打开保存文件的文件夹

使用方法:
1. 拖放文件到应用程序窗口或点击"选择文件"按钮添加文件
2. 点击"设置默认保存路径"按钮设置默认保存位置
3. 勾选要批量下载的文件，或使用"全选"按钮选择所有文件
4. 点击"批量下载"按钮将选中的文件保存到默认路径
5. 也可以点击单个文件的"保存"按钮选择不同的保存位置
