# ToBase64

这是一个简单的Flutter应用程序，用于将文件转换为Base64编码。

## 功能

- 选择任意文件并将其转换为Base64编码
- 复制Base64编码到剪贴板
- 将Base64编码保存为文本文件并分享

## 如何使用

1. 点击"选择文件"按钮选择要转换的文件
2. 应用程序会自动将文件转换为Base64编码
3. 转换完成后，您可以选择复制到剪贴板或保存并分享

## 安装

确保您的环境中已安装Flutter，然后运行：

```bash
flutter pub get
flutter run
```

## 依赖项

- file_picker: 用于选择文件
- path_provider: 用于获取应用程序文档目录
